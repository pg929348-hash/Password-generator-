# Password-generator-# 🔐 Python Password Generator

A simple Python program that generates random and strong passwords.  
It combines letters, numbers, and special characters to make secure passwords.

## ✨ Features
- Asks the user for password length
- Uses letters `a,b,c,d` + Numbers `1,2,3,4` + Symbols `$,#,~,!`
- Generates 6 different random passwords in one run
- Handles invalid input with error messages

## 🛠️ How to Use
1. Save the code as `password_generator.py`
2. Run it in Terminal or VS Code:
   ```bash
   python password_generator.py
